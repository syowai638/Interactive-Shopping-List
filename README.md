# Interactive Shopping List

An interactive shopping list application that allows users to:

- Add items to a shopping list
- Mark items as purchased
- Delete items
- Clear the entire list

## Features

1. **Dynamic List**: Items are dynamically added to the list with JavaScript.
2. **Event Handling**: Click events handle adding, deleting, marking as purchased, and clearing the list.
3. **Visual Indicators**: Purchased items are marked with a strikethrough and a green background.

### Bonus Features (Optional)
- **Edit List Items**: Allow users to edit existing items (not implemented yet).
- **Persistence**: Use `localStorage` to save the list across page reloads.
